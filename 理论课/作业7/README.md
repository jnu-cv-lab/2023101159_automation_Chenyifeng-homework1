# 传统机器学习手写数字分类实验
基于 sklearn digits 数据集，使用多种传统机器学习模型完成手写数字图像分类全流程实验

## 实验内容
- 数据集：sklearn 内置 8×8 手写数字数据集（1797 样本，0-9 共 10 类）
- 特征处理：将 8×8 图像展平为 64 维像素特征向量
- 数据划分：75% 训练集（1347 样本），25% 测试集（450 样本）
- 模型：KNN、朴素贝叶斯、逻辑回归、SVM、决策树、随机森林
- 评价指标：测试准确率、混淆矩阵、错误样本分析

## 实验结果
- SVM 准确率最高：**0.9800**
- 逻辑回归：0.9778
- 随机森林：0.9667
- KNN：0.9644
- 朴素贝叶斯：0.8289
- 决策树：0.9644

## 核心结论
- 传统机器学习需将图像转为固定长度特征向量才可输入模型
- 原始像素特征简单但对平移、旋转、缩放敏感
- SVM 在小样本图像分类上表现最优
- 随机森林比单棵决策树更稳定，泛化能力更强
- 错误分类多由数字形态相似、低分辨率细节丢失导致
