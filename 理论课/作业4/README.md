# 图像局部空间频率估计：FFT 与梯度法对比
简洁、精炼、可直接用作 `README.md`

---

## 项目简介
本项目实现两种图像局部空间频率估计方法，并进行定量对比与可视化分析：
- **FFT 95% 能量法**（频域金标准）
- **梯度幅值映射法**（空域快速估计）

通过平均绝对误差（MAE）与相关系数评估一致性，适用于图像下采样、抗混叠滤波、纹理分析等场景。

## 核心方法
### 1. FFT 95% 能量频率
- 图像分块 + 汉宁窗抑制频谱泄漏
- 计算二维 FFT 与功率谱
- 取累积能量达 95% 的径向频率作为局部频率
- 归一化至 [0, 0.5] 物理有效范围

### 2. 梯度法局部频率
- Sobel 算子计算梯度幅值
- 分块求取平均梯度强度
- 线性拟合映射到与 FFT 相同区间
- 无需频域变换，计算高效

### 评价指标
- **平均绝对误差 MAE**：全局估计偏差
- **皮尔逊相关系数**：分布线性一致性

## 实验结果
```
FFT 平均频率:      0.4995
梯度平均频率:      0.4994
平均绝对误差 MAE:  0.0011
相关系数:          0.1748
```

## 结果分析
- 全局均值几乎完全一致，**MAE 极低**，梯度法可有效替代 FFT 做全局频率估计
- 相关系数较弱，源于测试图像频率分布高度集中，局部差异小
- 梯度法计算更快、无需傅里叶变换，适合实时性场景
