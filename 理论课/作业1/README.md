## 项目概述
本作业围绕 OpenCV 库展开数字图像处理进阶实践，核心实现了彩色图像读取、YCbCr 色彩空间转换、Cb/Cr 通道下采样、插值恢复、图像重建、PSNR 质量评估及结果可视化对比等功能。通过本次作业，掌握 YCbCr 色彩空间的核心原理（亮度与色度分离），理解下采样/插值对图像质量的影响，熟悉 PSNR 指标在图像质量评估中的应用，为图像压缩、视频编码等进阶方向奠定基础。

## 技术栈
- 编程语言：Python 3.8+（兼容 3.7 及以上版本）
- 核心库：
  - OpenCV-Python（cv2）：图像读取、色彩空间转换、下采样/插值、图像保存
  - NumPy：图像像素数组操作、数值计算
  - Matplotlib：图像可视化对比展示
  - math：PSNR 指标数学计算
