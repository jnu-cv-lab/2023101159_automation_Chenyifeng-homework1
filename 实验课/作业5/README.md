# 数字图像处理
## 一、三类几何变换性质分析
对包含**外框矩形、圆形、网格平行线、垂直直线**的原始测试图，分别施加**相似变换、仿射变换、透视变换**，观察变换前后图像的几何特征，每类变换对几何性质的影响总结如下：

### 相似变换几何性质分析
直线是否保持为直线：保持
平行线是否仍保持平行：保持
两条垂直线是否仍垂直：保持
圆是否仍保持为圆：保持

### 仿射变换几何性质分析
直线是否保持为直线：保持
平行线是否仍保持平行：保持
两条垂直线是否仍垂直：保持
圆是否仍保持为圆：保持

### 透视变换几何性质分析
直线是否保持为直线：保持
平行线是否仍保持平行：保持
两条垂直线是否仍垂直：保持
圆是否仍保持为圆：保持

## 二、平面透视畸变图像校正实验
对桌面拍摄、存在透视畸变的带文字和表格的A4平面图像进行逆透视校正，对比原始畸变图与最终校正效果图，可以发现：
1.  校正整体结果较为满意，原图透视歪斜带来的文字拉伸畸变被大幅修复，校正后的文字形态清晰规整，基本还原纸张原生字体样貌，仅存在极微小的形变残留。
2.  原始图片里上窄下宽、梯形畸变的表格边框，校正后恢复为横平竖直的标准矩形边框，网格线条平直规整。
3.  本次校正输出画布严格遵循真实A4纸长宽比 $1:\sqrt{2}$，画面整体比例贴合真实纸张规格，文字与表格无明显拉伸、压缩畸变，平面结构还原效果良好。
