## 🧪 实验原理
### 1. 核心流程
`图像读入 → 两种下采样（直接/高斯滤波） → 三种插值恢复 → 空间域质量评估 → FFT频域分析 → DCT频域分析 → 结果可视化`

### 2. 关键技术
| 模块 | 核心方法 | 原理说明 |
|------|----------|----------|
| 下采样 | 直接下采样 / 高斯滤波+下采样 | 直接采样保留细节但易混叠；高斯滤波抑制混叠，图像更平滑 |
| 插值恢复 | 最近邻 / 双线性 / 双三次 | 最近邻计算快但锯齿明显；双线性平滑均衡；双三次细节保留最优 |
| 空间域评价 | MSE / PSNR | MSE越小、PSNR越大，恢复质量越好 |
| FFT分析 | 二维傅里叶变换+频谱中心化 | 低频对应图像结构，高频对应边缘细节，直观展示细节丢失 |
| DCT分析 | 二维DCT+低频能量占比 | 能量集中在左上角低频，占比越高，图像越平滑 |

## ✅ 实验结论
1.  **下采样优化**：高斯预滤波可有效抑制混叠失真，提升视觉质量，但会轻微降低PSNR
2.  **插值性能排序**：双线性 ≈ 双三次 > 最近邻，双线性综合性价比最高
3.  **不可逆性验证**：下采样会不可逆丢失高频细节，插值仅能近似恢复低频结构
4.  **频域量化**：DCT低频能量占比可量化图像平滑度，为算法选择提供依据

## 🐛 常见问题解决
- **文件找不到**：使用绝对路径替换相对路径，确保图像文件存在
- **Matplotlib无法显示**：代码已默认使用`plt.savefig()`保存图片，无需弹窗
- **混叠失真明显**：增大高斯滤波核大小（如`ksize=(7,7)`）增强平滑效果
